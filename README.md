# -Flask-Mini-Web-App
 Create a small Flask app (todo, notes, contact form). Requirements: routing, templates, optional CRUD,
Bootstrap UI. Below is a compact Flask todo app with in-memory storage (for demo). For production use a
database (SQLite).
Approach: - Small Flask app with routes: index (list todos), add, delete. - Use simple Bootstrap template. - Provide
instructions for running and optional deployment notes

Run instructions: 1. Install dependencies: pip install flask pandas requests 2. For Task 1,2,3: run the python
scripts as shown above. 3. For Task 4: - Create a folder structure: /app.py /templates/index.html - Run: flask run
(or: python app.py) - Open http://127.0.0.1:5000
